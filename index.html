Step 1 of 2 · One-time setup

Connect your Ledger

Your budget lives in your own free Firebase project — private to you, synced across every device you log into.

1
Go to console.firebase.google.com → create a project (any name).
2
Build → Firestore Database → Create database → start in test mode.
3
Build → Authentication → Sign-in method → enable Email/Password.
4
Project settings → your apps → Add app → Web (</>) → copy the firebaseConfig object and paste it below.

This is stored only in this browser's local storage and sent straight to Google's Firebase — it never passes through anyone else's server. Each device needs this pasted in once.
